# AI Digest: April 2026

**TL;DR:** The industry is shipping agentic systems at scale—personal knowledge bases replacing monolithic apps, LRMs proving base LLMs wrong about reasoning, and robots getting real-time agency. Policy makers are catching up to prevent a patchwork of state regulations, while anthropic tightens third-party tool access.

## Personal Knowledge Systems & Agent-Native Architecture

[LLM Knowledge Bases as Explicit Personal Wikis](https://x.com/karpathy/status/2040572272944324650) — Karpathy outlines why personal wikis beat implicit AI memory: explicit/inspectable, user-owned data in universal formats, BYOAI flexibility, and full control over your information artifact rather than vendor lock-in.

[GitHub Gists as Superior Discussion Medium](https://x.com/karpathy/status/2040806346556428585) — Markdown format and lack of engagement incentives produce better technical discussion than X; suggests platforms reconsider what drives quality conversation.

[Agent-First "Idea Files" Over Code Sharing](https://x.com/karpathy/status/2040470801506541998) — In the LLM agent era, sharing abstract problem descriptions beats sharing specific implementations—let agents customize solutions for individual needs.

[Agent Memory Systems Now Teachable](https://x.com/AndrewYNg/status/2034314027678192114) — New course covers persistent memory stores, memory managers, and semantic tool retrieval so agents learn across sessions rather than resetting.

## Intelligence & Reasoning Breakthroughs

[LRMs Finally Demonstrate Reasoning Base LLMs Lacked](https://x.com/fchollet/status/2041248358518731020) — Latest reasoning models solve math/generalization problems that stumped base LLMs; Chollet notes the 2024 "LLMs can already reason" crowd missed what to look for.

[Base LLMs Have Zero Fluid Intelligence—Confirmed](https://x.com/fchollet/status/2041249002835128529) — No longer controversial: base LLMs from 2023-24 couldn't reason; people advocating otherwise simply didn't know what to measure.

## Agent Economics & Organizational Design

[Agent Stack Mirrors Organizational Economics](https://x.com/emollick/status/2041165222438711320) — High-ability agents are expensive, low-ability cheap; delegation strategy, incentive alignment, and handoff costs matter as much in agentic systems as human orgs.

[2025 Had No Real GenAI Work Impact Yet](https://x.com/emollick/status/2041218593367478276) — No agentic tools existed, adoption was slow, everyone experimented. Real disruption starts 2027 when studies from this year won't apply.

[Agent API Documentation Hub Gains Traction](https://x.com/AndrewYNg/status/2033577583200354812) — Context Hub (chub) hit 6K GitHub stars; agents now share feedback on API docs to collectively improve them—a Stack Overflow for coding agents.

## Policy & Regulation

[Andrew Ng Warns Against Anti-AI Propaganda Campaigns](https://x.com/AndrewYNg/status/2039051446084153834) — Organizations deliberately testing which AI fear messages resonate; messaging shifts from extinction to warfare and environment when extinction fails. Federal preemption framework beats patchwork state rules.

[Anthropic Engages with Department of War](https://x.com/DarioAmodei/status/2027152488659394660) — Dario Amodei's statements on discussions with DoW; AI security enters hard defense territory.

[Anthropic Essays AI Risk to Democracy & Economy](https://x.com/DarioAmodei/status/2015833046327402527) — "The Adolescence of Technology" positions powerful AI as national security risk across economic and democratic domains.
