# AI Digest: March 17, 2026

**TL;DR:** GPT-5.4 dominates builder adoption with 5T tokens/day in a week; Claude Code shifts agents from assistants to team operators with folder-based skills architecture; Yann LeCun raises $1.03B for world models as the anti-LLM bet; production agents expose infrastructure gaps—prefilling changes break stacks, memory management remains unsolved.
