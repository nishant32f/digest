# AI Capability Gaps, Agents Go Wild, and the Cyber Security Reckoning

**TL;DR:** There's a massive disconnect between casual AI users (seeing ChatGPT fumble voice queries) and power users (watching agents autonomously refactor codebases for hours). The real breakthroughs are in technical domains with verifiable rewards—coding, math, security—while general tasks lag. Meanwhile, agentic models are now operating GUIs faster than humans, doubling agent usage, and Anthropic is weaponizing AI to find zero-days before bad actors do.

## Model Capability & Reality

[There's a growing gap in understanding of AI capability—free tier vs frontier models are completely different beasts](https://x.com/karpathy/status/2042334451611693415) — Karpathy's essential breakdown: ChatGPT free tier fumbles dumb questions while paid Codex/Claude Code spend an hour coherently restructuring entire codebases. The gap exists because technical domains have explicit reward functions (tests pass/fail) that reinforce learning works on, unlike writing which is subjective.

[The OpenClaw moment went viral because non-technical people finally saw real agentic models, not just ChatGPT](https://x.com/karpathy/status/2042341482531864741) — Most people only knew AI as a website until agents showed what's actually possible.

[LLMs are moving from code manipulation to knowledge base building](https://x.com/karpathy/status/2040470801506541998) — Karpathy's sharing an "idea file" concept: instead of shipping specific code, you share an abstract idea and let agents customize it for your needs. This is the future workflow.

[Claude Opus 4.7 launched with major improvements and increased rate limits](https://x.com/bcherny/status/2044839936235553167) — Uses more thinking tokens but Anthropic compensated with higher rate limits for subscribers to balance it out.

## Agent Autonomy Reaches New Level

[LLMs now operate GUIs as fast as humans, and it's genuinely surreal to watch](https://x.com/sama/status/2044858929491202435) — Sam Altman retweeting the moment when agent speed became humanlike; this is the inflection point for real assistant adoption.

[Codex can use all Mac apps in parallel without interfering with your work](https://x.com/sama/status/2044858862042591378) — Computer use isn't just possible; it's genuinely useful. Agents learn from experience and proactively suggest tasks they can handle.

[Cognition agent usage has doubled globally since Devin launched](https://x.com/swyx/status/2043770029024653798) — Devin's been growing 50%+ month-over-month. People are finding creative use cases when you can compose agents together and make them proactive.

[Agent recursion—agents managing agents—is the real capabilities frontier](https://x.com/swyx/status/2044723726755955128) — Managing subagents is optimization; having agents boss other agents is a capabilities leap. This is how you scale beyond single-agent limits.

## Specs, not code, are the new interface

[Spec-driven development with coding agents is the skill that matters now](https://x.com/AndrewYNg/status/2044449830605582629) — Andrew Ng's new course teaches you to write detailed specs that control large code changes with a few words. Vibe coding is fast but wrong; specs let you stay in control as complexity grows.

[The "Product Management Bottleneck" is real—deciding what to build matters more than building it](https://x.com/AndrewYNg/status/2043742105852621052) — As coding becomes frictionless, software engineering job postings are *rising*, not falling. Skills shifting: specs, architecture decisions, and "what to build" now trump raw coding ability.

## Cyber Security & AI Vulnerabilities

[Anthropic launched Project Glasswing to find software vulnerabilities at scale](https://x.com/DarioAmodei/status/2041580334693720511) — Powered by Claude Mythos Preview, which finds bugs better than all but the most skilled humans. This is an arms race: AI finding zero-days before attackers do.

[Cyber is the first clear and present danger from frontier AI, but won't be the last](https://x.com/DarioAmodei/status/2041580343472337145) — Dario Amodei framing this as a blueprint for confronting even harder challenges ahead. If we get this right, we get a more secure internet. If we don't, well.

## Capability Reality Checks

[Simple methods don't scale; scalable methods are complex](https://x.com/fchollet/status/2044695036470689971) — François Chollet dismantling the misconception that simplicity enables scale. SVMs and random forests are simple but don't scale. Transformers are 10x more complex but scale like crazy.

[Local Qwen 35B on a laptop beats Claude Opus 4.7 on some visual tasks](https://x.com/simonw/status/2044830134885306701) — Simon Willison's pelican and flamingo-on-unicycle benchmarks show frontier models aren't unbeatable at everything. Local models are catching up on specific tasks.

[Opus 4.7's thinking mode triggers are oddly specific](https://x.com/emollick/status/2044863531900686775) — Ethan Mollick found asking for a sestina (obscure poetic form) triggers safety guardrails. Model weirdness is still alive and well, even at the frontier.
