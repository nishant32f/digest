# AI Digest: The Pragmatic Era Emerges

**TL;DR:** The AI industry is crystallizing around agent-native architectures and user control—LLM-powered knowledge systems, persistent agent memory, and open data formats are replacing monolithic AI apps. Meanwhile, infrastructure scale accelerates (Anthropic hits $30B run-rate), while regulatory and workplace design questions mount.

---

## Knowledge & Personalization

[LLM Knowledge Bases as Personal Wikis](https://x.com/karpathy/status/2039805659525644595) — Andrej Karpathy advocates building explicit, user-owned knowledge repositories instead of relying on proprietary AI memory, shifting tokens toward knowledge manipulation over code work.

[Farzapedia: A Personal Wikipedia](https://x.com/FarzaTV/status/2040563939797504467) — Real implementation of wiki-as-knowledge-artifact using diary entries and messages; Karpathy elaborates why file-based, format-agnostic storage beats proprietary black-box memory systems.

[Agent Memory Courses Build Persistence](https://x.com/AndrewYNg/status/2034314027678192114) — Andrew Ng launches structured training on semantic memory managers that let agents retain and refine knowledge across sessions, enabling long-horizon reasoning.

[GitHub Gists Outperform X as Discussion Forums](https://x.com/karpathy/status/2040806346556428585) — Karpathy observes markdown gists attract more thoughtful commentary than algorithmic social feeds, suggesting format and incentives matter more than reach.
